Zeichnungsfeld

## Programmieren Im Zeichungsfeld kannst du zeichnen, indem du mit <kbd>F12</kbd> die Konsole öffnest und dort die Befehle `right()`, `left()`, `up()` und `down()` eingibst. Der Code ```js right() up() left() down() ``` zeichnet zum Beispiel ein Quadrat. Mit dem Befehl `goTo()` kannst du an einen Ort springen, ohne zu zeichnen: ```js goTo(20, 40) ``` geht zur Position (20,40), ohne eine Linie zu zeichnen. In deinem Code kannst du auch Variablen verwenden. In Variablen kannst du Zahlen speichern, und diese später wieder verwenden. Variablen sind also wie Platzhalter. Um eine Zahl zu speichern kannst du diesen Code verwenden: ```js var x = 20 ``` Dies speichert die Zahl 20 in der Variable x. Um dann die Variable zu verwenden, kannst du den Namen der Variable in den Code einsetzen: ```js up(x) left(x) down(x) right(x) ``` Dies zeichnet ein Quadrat mit Seitenlänge 20, weil in `x` die Zahl 20 gespeichert ist ### Aufgaben * Zeichne ein Quadrat, indem du eine Variable verwendest ![quadrat](images/square.png) Du kannst Code auch automatisch wiederholen lassen. Dazu wird meist eine **Schleife** verwendet. Mit einer Schleife wird eine Variable automatisch hochgezählt, und der Code jedes mal ausgeführt: ```js for(var x = 1; x < 40; x = x + 1) { up(x) left(x) } ``` ### Aufgaben * Zeichne das folgende Muster ![quadrat](images/teeth.png) * Zeichne die folgende Treppe ![quadrat](images/stairs-small.png) * Zeichne die folgende Treppe ![quadrat](images/stairs-expo.png)
